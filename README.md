# india-runs-hackathon-ai-recruiter
AI-powered candidate discovery and ranking system that evaluates skills, experience, recruiter engagement, and behavioral signals to identify the best candidates for a given job description.
TalentIQ AI is an intelligent candidate ranking platform developed for the India Runs Data & AI Challenge. The system processes large-scale candidate datasets and generates recruiter-ready shortlists by analyzing technical skills, professional experience, endorsements, profile quality, recruiter engagement metrics, and job-description relevance. Unlike traditional keyword-based applicant tracking systems, TalentIQ AI uses a multi-signal scoring approach to identify candidates who are genuinely suitable for the role, providing transparent and explainable rankings that recruiters can trust.
